Project 1: Loops, Conditionals, Variables. Students will learn about three fundamental concepts: variables, conditionals, and loops. Variables are essential for storing and manipulating data within the program, with each variable having a name and a type that defines the type of data it can hold, such as numbers or text. Conditionals allow the program to make decisions based on certain conditions, using structures such as if, else, and else if to execute different blocks of code depending on whether a condition is true or false. Loops are powerful tools that allow actions to be repeated multiple times, with the most common loops being for, while, and do-while, which execute a block of code repeatedly until a specific condition is met. These concepts are integrated to create efficient and functional programs.

Project 2: Advanced Conditionals, Nested Loops, Functions. Students will learn to handle advanced conditionals, using logical operators (AND, OR, NOT) and combinations of conditions to make complex decisions in code. They will also explore nested loops, allowing iterations within other iterations to work with more complex data structures, such as arrays. Finally, they will focus on functions, learning to define and use reusable blocks of code that improve program modularity, readability, and maintainability. These topics will enable students to develop more sophisticated and efficient programs.

Project 3: Inheritance, Advanced Loops. Students will explore inheritance and advanced loops. Inheritance is a pillar of object-oriented programming that allows new classes to be built on top of existing classes, facilitating code reuse and the creation of class hierarchies. Advanced loops will cover more complex techniques and structures, optimizing iteration over data sets and allowing for greater flexibility and efficiency in handling repetitive operations. These advanced concepts will equip students with tools to build more robust and scalable programs.
